My IP — WebRTC exposure demo

Collects the addresses WebRTC discloses through ICE candidates, classifies each (private / link-local / CGNAT / loopback / ULA vs public), and renders the leak verdict against the public IP you paste in (copy it from any "what is my IP" page). With STUN enabled the probe's only network traffic is one STUN binding request; untick it for a fully local probe. Needs a browser with WebRTC.
